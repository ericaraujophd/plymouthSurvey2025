## ‚ùì Common Questions

**Q: Do I need to read everything?**
A: No. Start with executive summary. Go deeper as needed for your role.

**Q: Can I trust this analysis?**
A: Yes. All statistics and quotes come directly from the 68 survey responses. No data was invented or skewed.

**Q: What if I disagree with something?**
A: Good! This represents congregation's voice. Leadership should add wisdom and context. Use as input, not dictation.

**Q: Where's the raw data?**
A: [survey-answers.csv](survey-answers.csv) - But don't modify it (per project requirements)

**Q: Can I see the form we need to fill out?**
A: [church-profile-form.md](church-profile-form.md)

**Q: How were word clouds created?**
A: Text analysis of open-ended responses. Bigger words = mentioned more frequently.

**Q: Can we rerun the analysis if we get more responses?**
A: Yes. Scripts are in [analysis/](analysis/) folder. Instructions in [README.md](README.md).
